# MentalHealthApp 🌿

Мобильное приложение для мониторинга ментального здоровья с использованием Java, MVVM и Supabase.

##  Основные функции
* **Трекинг настроения**: выбор эмоционального состояния на календаре с защитой от дубликатов (Upsert).
* **Личные заметки**: ведение дневника с поддержкой форматирования текста (Bold/Italic).
* **AI-Ассистент**: встроенный чат для психологической поддержки.
* **Аналитика**: визуализация динамики состояния за неделю на графиках.
* **Безопасность**: авторизация через JWT и защита данных на уровне строк (RLS) в PostgreSQL.

##  Стек технологий
* **Язык**: Java 11 (Android SDK)
* **Архитектура**: MVVM + Repository Pattern
* **Бэкенд**: Supabase (Auth, Database, Storage)
* **Библиотеки**: Retrofit 2, OkHttp, GSON, MPAndroidChart, JUnit 4

##  Установка и запуск
1. Клонируйте репозиторий.
2. Настройте `local.properties`, указав ваши `SUPABASE_URL` и `SUPABASE_ANON_KEY`.
3. Соберите проект в Android Studio и запустите на эмуляторе (API 24+).
